# 🛍️ PetSupplies Shop - 宠物用品全栈电商平台

一个基于 React + Node.js 构建的现代化全栈 B2C 电商平台，包含完整的用户认证、商品管理、购物车和订单流程。

**前端+后端源码:** 🔗 [https://github.com/codeDancer04/pet-supplies-shop](https://github.com/codeDancer04/pet-supplies-shop)

## ✨ 项目特色与亮点

### 🚀 技术亮点
- **全栈 TypeScript**: 前后端均使用 TypeScript 开发，提供严格的类型安全保证
- **工程化实践**: 基于 Vite 构建，集成 ESLint + Prettier，实现高效的开发体验
- **组件抽象**: 封装 8+ 高复用性业务组件，支持 UI 与逻辑分离
- **状态管理**: 使用 React Hooks 实现复杂状态逻辑的优雅管理

### 💡 业务功能
- **用户系统**: JWT 鉴权、注册登录、权限控制
- **商品系统**: 商品展示、添加至购物车
- **购物流程**: 购物车管理、订单生成、流程可视化

## 🛠 技术栈
