## ✨ 核心亮点：智能购物 Agent

本项目不仅仅是一个传统的电商平台，核心在于集成了一个**能够“听懂”并“执行”用户指令的智能体**。

- **🧠 深度意图识别**: 基于大语言模型（Qwen-plus）精准理解复杂的自然语言指令，如“帮我查一下最近的订单”、“我想买两袋价格在50元左右的狗粮”。
- **🛠️ 全自动工具调用 (Function Calling)**:
  - **商品检索**: 智能体可根据模糊描述（价格区间、类别、关键词）查询数据库。
  - **订单管理**: 支持自然语言查询历史订单状态、金额及详情。
  - **语音下单**: 实现了**对话即下单**的完整流程，用户只需说出“帮我下单这件商品”，智能体即可自动校验库存、计算金额并生成订单。
- **🔐 安全的上下文交互**: 智能体与 JWT 鉴权系统深度打通，确保只有在登录状态下才能访问敏感数据（如个人订单），并能根据用户身份提供个性化服务。

## ✨ 更多项目特色

### 🚀 技术亮点
- **全栈 TypeScript**: 前后端均使用 TypeScript 开发，提供严格的类型安全保证
- **工程化实践**: 基于 Vite 构建，集成 ESLint + Prettier，实现高效的开发体验
- **组件抽象**: 封装 8+ 高复用性业务组件，支持 UI 与逻辑分离
- **状态管理**: 使用 React Hooks + Context/Redux 实现复杂状态逻辑的优雅管理

### 💡 业务功能
- **用户系统**: JWT 鉴权、注册登录、权限控制
- **商品系统**: 商品瀑布流展示、搜索过滤、详情页
- **购物流程**: 购物车管理、订单生成、支付模拟、流程可视化
